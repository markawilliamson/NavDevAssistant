# Changelog

## 1.0.2 - 2026-07-01

- Added field-level `FieldClass` and `CalcFormula` metadata to the persistent JSON symbol cache, including support for multi-line FlowField formulas.
- Show FlowField metadata in field hover text, document symbols, and field completion documentation.
- Added repository maintenance instructions in `AGENTS.md`, including validation and changelog maintenance guidance.

## 1.0.0 - 2026-06-12

- Initial public release of NAV Dev Assistant.
- Adds NAV object syntax highlighting, workspace indexing, object navigation, dependency views, and AI-assisted code explanations.
